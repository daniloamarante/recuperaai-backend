# RecuperaAI – Backend

Plataforma inteligente para recuperação de carrinhos abandonados com campanhas automatizadas via e-mail e WhatsApp.

## Instalação

```bash
npm install
npx prisma generate
npx prisma migrate dev --name init
npm start
```

## Variáveis de Ambiente

Crie um arquivo `.env` baseado em:

```
PORT=4000
```
